# Passo a passo - Fase 1.0 - Mockup navegável

## Objetivo

Registrar e validar o estado atual do mockup navegável como base da evolução do projeto.

## Etapa 1 - Confirmar estrutura atual

### Descrição

Verificar se o projeto possui as telas, scripts, estilos e documentos esperados para o MVP navegável.

### Ações necessárias

- Listar arquivos de `cpa-admin`.
- Listar arquivos de `cpa-docs`.
- Confirmar presença das telas principais.
- Confirmar presença de `design-system.html`.

### Arquivos envolvidos

- `cpa-admin/index.html`
- `cpa-admin/compras.html`
- `cpa-admin/revisao.html`
- `cpa-admin/comparacao.html`
- `cpa-admin/detalhes.html`
- `cpa-admin/design-system.html`
- `cpa-docs/index.md`

### Critério de conclusão

As telas e documentos principais existem e representam o fluxo atual.

## Etapa 2 - Registrar capacidades existentes

### Descrição

Documentar o que o mockup já demonstra sem tratar essas capacidades como sistema real.

### Ações necessárias

- Confirmar fluxo de compra.
- Confirmar busca, filtros, similares, revisão, comparação e detalhes.
- Confirmar uso de dados mockados.
- Confirmar design system vivo.

### Arquivos envolvidos

- `cpa-docs/screens/`
- `cpa-docs/flows/`
- `cpa-docs/business-rules.md`
- `cpa-docs/design-system.md`

### Critério de conclusão

A fase 1.0 está descrita como protótipo navegável validável.

## Etapa 3 - Registrar pendências

### Descrição

Listar o que ainda não existe e não deve ser presumido.

### Ações necessárias

- Registrar ausência de Git.
- Registrar ausência de backend, API, banco, autenticação, crawlers, testes e análise de dados.
- Marcar auditoria e QA como próximos passos.

### Arquivos envolvidos

- `cpa-docs/desenvolvimento_fases/fase-1.0-mockup-navegavel.md`

### Critério de conclusão

As limitações da fase 1.0 estão claras e não confundem mockup com produto real.

## Checklist final da fase

- [ ] Estrutura atual validada.
- [ ] Telas principais reconhecidas.
- [ ] Dados mockados identificados.
- [ ] Documentação inicial reconhecida.
- [ ] Pendências estruturais registradas.
- [ ] Fase 1.1 definida como próximo passo.
