Esta pasta funciona como guia operacional para executar cada fase de evolução do CPA Mockup / Comparaqui.

- Quais arquivos revisar.
- Qual ordem seguir.
- Quais critérios usar para considerar cada etapa concluída.
- Quais checklists validar antes de avançar.
