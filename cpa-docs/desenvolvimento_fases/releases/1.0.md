# Release v1.0 - CPA Mockup / Comparaqui

## Data

2026-06-28

## Resumo

Esta release congela o mockup navegavel do CPA Mockup / Comparaqui como base estavel para a Fase 1.1 - Consolidacao do Mockup.

## Escopo incluido

- Telas HTML do mockup em `cpa-admin`.
- Scripts, estilos e assets locais usados pelo mockup.
- Documentacao de produto, telas, fluxos, componentes e fases em `cpa-docs`.

## Escopo excluido

- Backend.
- APIs reais.
- Banco de dados.
- Crawlers.
- Migracao para Next.js ou NestJS.
